# 智慧图书馆座位预约系统（PC 端）

基于 `Vue 3 + Vite + Element Plus + Pinia + Vue Router` 实现的图书馆座位预约前端项目，包含登录注册、座位地图、首页预约、预约管理、积分体系、身份绑定、个人中心等功能。

## 项目简介

本项目面向图书馆用户端场景，提供完整的座位预约前端交互流程：

- 登录 / 注册
- 身份绑定（学号绑定）
- 首页预约
- 独立座位地图页
- 我的预约与历史记录
- 积分规则与积分记录
- 收藏、通知、设置、帮助反馈

当前项目以前端演示与交互实现为主，账号注册信息主要写入本地存储，部分接口通过 mock 方式模拟。

## 技术栈

- Vue 3
- Vite
- Element Plus
- Pinia
- Vue Router
- Axios
- MockJS

## 安装与运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发环境

```bash
npm run dev
```

默认访问地址：

```text
http://localhost:5173/
```

### 3. 打包构建

```bash
npm run build
```

### 4. 本地预览构建结果

```bash
npm run preview
```

## 当前功能说明

### 1. 登录与注册

- 支持登录 / 注册切换
- 新注册账号会保存到本地存储
- 登录后如果未绑定学号，会引导进入身份绑定页

### 2. 身份绑定

- 支持填写真实姓名、学号 / 工号、校园卡密码
- 支持暂不绑定，先进入首页
- 绑定成功后直接进入首页

### 3. 首页预约

- 首页保留原有预约方式
- 先选座位，再选择日期与时间段
- 支持二次确认弹窗
- 展示预计积分变动

### 4. 座位地图

- 独立座位地图页 `/seat-map`
- 地图布局参考旧版本地图设计
- 所有楼层统一使用同一套地图编码和区域划分
- 可从地图页选择座位并回到首页完成预约

### 5. 预约规则

当前页面中已实现的预约限制包括：

- 仅支持预约今天起 7 天内的时段
- 结束时间必须晚于开始时间
- 开始时间必须晚于当前时间
- 与已有预约时间冲突时不可重复预约
- 单次最长时段说明保留在首页预约区域

### 6. 积分体系

- 当前采用 100 分制基础分
- 支持超过 100 分
- 按时签到：`+3`
- 爽约：`-15`
- 已实现积分规则页与积分记录页展示

## 说明

- 本项目当前以前端展示与业务流程模拟为主
- 部分数据来自 mock，部分账号数据来自浏览器本地存储
- `.env.development` 等本地环境文件不建议直接提交到公开仓库
